## Projects
# Project 1
- Analysed the relationship between short-term monetary policy and poverty in Nigeria using time-series regression in R.
- Built an initial OLS model and improved it by addressing serial correlation with the Cochrane-Orcutt procedure.
- Used diagnostic tests including RESET, Breusch-Pagan, Durbin-Watson, PACF, and VIF to assess model assumptions.
- Interpreted regression outputs to evaluate the effects of inflation, policy timing, and poverty persistence.
- Found that short-term policy did not have a statistically significant effect on poverty reduction.

# Project 2
- Analysed 532 students’ exam performance using multiple linear regression in R.
- Explored the data through summary statistics, visualisations, and correlation analysis.
- Refined the model by removing insignificant variables and checking residual diagnostics and multicollinearity.
- Identified previous exam scores, study hours, homework completion, and study group participation as the most useful predictors.
- Used the final model to generate predicted exam scores for different student scenarios.

# Project 3
- Analysed development indicators for 120 countries to identify natural country groupings.
- Used PCA to reduce correlated variables and clarify the main dimensions of development.
- Applied both K-Means and hierarchical clustering to create and validate a three-cluster solution.
- Interpreted the clusters as low-, middle-, and high-development profiles based on health, economic, and trade indicators.
- Showed that similar health spending levels can still produce very different outcomes, suggesting that efficiency and institutions matter as well as funding.
